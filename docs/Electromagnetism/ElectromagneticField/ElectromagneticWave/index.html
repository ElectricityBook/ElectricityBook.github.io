전자파

전자파의 특징#

  1. 전계와 자계가 직각으로 진동한다.
  2. 전계와 자계의 위상은 동일하다.
  3. 전자파의 전달 방향은 E×B\mathbf{E}\times\mathbf{B}E×B이다
  4. 전계와 자계의 비는 다음과 같다.
    EB=μϵ\frac{E}{B}=\sqrt{\frac{\mu}{\epsilon}}BE​=ϵμ​​

전자파의 속도#

v=fλ=1ϵμv=f\lambda = \frac{1}{\sqrt{\epsilon\mu}}v=fλ=ϵμ​1​
